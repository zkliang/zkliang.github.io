# 广告联盟申请清单（静态站适用）

> 适用：流量起来后接展示广告（AdSense / 百度联盟等）。
> 新站建议：先用 CPS（无需审核、即时上线），等有一定流量再接展示广告。

---

## 一、可选联盟对比

| 联盟 | 门槛 | 结算 | 适合阶段 |
|------|------|------|----------|
| Google AdSense | 需合规隐私页 + 内容原创 + 一定流量 | 美元 | 合规 + 有流量后 |
| 百度联盟 | 通常需 ICP 备案 | 人民币 | 国内主流 |
| 阿里妈妈（淘宝联盟） | 低 | 人民币 | 电商 CPS |
| 京东联盟 | 低 | 人民币 | 3C CPS |
| 360 联盟 | 中 | 人民币 | 备选 |

---

## 二、申请前准备（打勾核对）

- [x] 隐私政策披露广告（privacy.html 已含广告 / 跟踪披露）✅
- [x] 合法内容（开源工具导航，无违规内容）✅
- [ ] ICP 备案（国内联盟通常需要；freenav.net 需自行备案）
- [ ] Cookie / 跟踪同意横幅（GDPR + 国内合规建议加）
- [ ] 独立域名已生效（freenav.net，DNS 配置中）
- [ ] 站点稳定运行、无大量外链失效

---

## 三、接入方式（纯静态站）

1. 注册联盟，获取广告 JS 代码（一段 `<script>`）。
2. 在 `index.html` / 栏目页合适位置插入广告容器 `<div>` + 联盟 JS。
3. 建议广告位：
   - 首页底部
   - 分类页侧栏 / 底部
   - 工具详情 / 文章页底部
4. 控制密度，每页 1–2 个，避免影响体验。

---

## 四、合规要点

- 隐私政策已披露「使用广告联盟及跟踪技术」✅（见 privacy.html）。
- 建议在页脚加「广告」标识。
- 国内运营建议办理 ICP 备案（freenav.net 域名备案，需服务器空间）。
- 展示广告 + CPS 并存时，CPS 链接仍需单独标注「推广 / 赞助」。

---

## 五、冷启动路径（推荐顺序）

```
第 1 步（现在）：接 CPS —— 淘宝/京东/云联盟，挂链接即上线，0 审核
   ↓
第 2 步（≥ 300 UV/天）：申请 AdSense / 百度联盟展示广告
   ↓
第 3 步（≥ 稳定流量 + 媒体包）：启动招商赞助（见 01 模板）
```

这样从第一天就有变现可能，且不依赖谈判或审核。
