# 阿里云大使（云大使）— 拿专属推广链接

> 这是目前最推荐给 FreeNav 的 CPS 路径：不验证网站、不要求备案、个人可加入、和开发者/站长受众最搭（卖服务器/域名/CDN，新用户首单返佣最高 30%）。
> 相比淘宝联盟（要 root.txt 网站验证 + ICP 备案号，GitHub Pages 国内抓不到），云大使完全没有这个坑。

## 一、入口

- 招募页：https://dashi.aliyun.com/
- 管理后台：https://dashi.aliyun.com/personal-center/profile
- 推广素材（拿链接的地方）：https://dashi.aliyun.com/personal-center/materials/popular-materials

## 二、拿链接步骤（约 3 分钟）

1. **登录**：用你买 `freenav.net` 的那个阿里云账号直接登录 `dashi.aliyun.com`（不用重新注册）。
2. **签协议**：首次进入会让你签一份电子合作协议，勾选同意即可。
3. **进推广素材**：左侧「推广素材」→「官方热门素材」。
4. **选活动**：挑一个和受众匹配的活动，推荐：
   - 「云服务器 ECS 新购 / 续费」（开发者装机刚需，转化高）
   - 「域名注册」（你本来就在卖域名认知，顺手推）
   - 「新人特惠 / 上云礼包」（拉新返佣比例最高）
5. **复制链接**：点对应活动的 **「复制链接」** 按钮。
   - 生成的链接会自动带上你的专属推广码，形如：
     `https://www.aliyun.com/daily-act/...?userCode=XXXXXX`
   - 这个 `userCode=XXXXXX` 就是你的返佣身份证，**必须带**，否则不算你的单。
6. **粘贴到站点**：把链接填进 `assets/js/config.js` 的 `FREENAV_AFFILIATE` 数组（见下方）。
7. **验证有效**：下单前回后台「推广管理」看一下是否显示有效关联，避免链接失效。

## 三、填进 FreeNav 站点

`assets/js/config.js` 里已经预留好插槽，照这样加一条：

```js
window.FREENAV_AFFILIATE = [
  {
    name: "阿里云服务器 ECS",
    note: "新用户首单返佣高，站长装机刚需",
    url: "https://www.aliyun.com/...?userCode=你的码",
    tag: "推广"
  }
];
```

填完 `git add . && git commit && git push`，首页底部「编辑推荐」区会自动出现带「推广」徽章的卡片，合规带 `rel="sponsored"`。

## 四、返佣 & 提现

- 返佣比例：按等级 15%–30%（新用户/拉新活动比例更高）。
- 结算：订单完成后次月结算，后台「收益」可见。
- 提现：绑定支付宝，达到门槛即可提现到你的支付宝（和买域名的账号同一实名）。

## 五、为什么这条路比淘宝联盟稳

| 对比项 | 淘宝联盟 | 阿里云大使 |
|--------|----------|------------|
| 网站验证 | 要 root.txt（GitHub Pages 国内抓不到 ❌） | 不需要 ✅ |
| ICP 备案号 | 必填 | 不需要 ✅ |
| 个人可加入 | 可 | 可 ✅ |
| 受众匹配 | 泛电商 | 开发者/站长 精准 ✅ |
| 返佣 | 商品佣金 | 最高 30% + 拉新奖励 ✅ |

> 结论：淘宝联盟那个表单先放着（或填「无」碰运气），主力走阿里云大使。等 `freenav.net` 自定义域名 DNS 全球生效后，回头把推广域名从 `zkliang.github.io` 换成 `freenav.net` 即可。
