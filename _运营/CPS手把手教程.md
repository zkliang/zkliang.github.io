# CPS 手把手教程（FreeNav 专用，零基础版）

> 你刚才说"不会 CPS"——看完这篇你就能上手。全程不需要写代码、不需要后端。
> CPS 一句话：你放一个**带返佣的专属链接**，别人通过它**买了东西**，平台自动给你佣金。

---

## 一、CPS 到底怎么赚钱（30 秒看懂）

```
你站点上的推广链接（带你的专属 ID）
        ↓ 用户点击
跳到淘宝 / 京东 / 阿里云 等平台
        ↓ 用户下单付款
平台自动记录"这是 FreeNav 带来的单"
        ↓
佣金结算到你的联盟账户（按月提现到支付宝/银行卡）
```

你不用管发货、客服、退货——全是平台和商家的事。你只负责"放链接"和"有人点"。

---

## 二、先从哪个平台开始？（推荐：淘宝联盟）

理由：**门槛最低、手机就能搞定、当天可上线**。

| 平台 | 门槛 | 适合你站点的品类 | 备注 |
|------|------|------------------|------|
| **淘宝联盟** ✅ 首选 | 手机号 + 支付宝实名，0 费用 | 数码、办公、课程、软件周边 | 手机 App 就能操作 |
| 阿里云推广 | 阿里云账号 + 实名 | 服务器、域名、CDN | 新用户首单返佣高，和站长受众绝配 |
| 京东联盟 | 京东账号 + 实名 | 3C、云主机、外设 | 同理 |

> 建议：先只做**淘宝联盟**一个，跑通流程再说。别一上来全注册。

---

## 三、淘宝联盟注册 + 拿链接（一步步）

### 第 1 步：注册
1. 手机应用商店搜 **「淘宝联盟」** App，下载。
2. 用**淘宝 / 支付宝账号**登录（没有就注册一个）。
3. 按提示完成**实名认证**（绑定支付宝，刷脸，1 分钟）。

### 第 2 步：创建"推广位"（重要）
1. 打开 App → 点底部 **「我的」** → **「推广位管理」**。
2. 新建一个推广位，名称随便填，比如 `FreeNav首页`。
3. 推广位类型选 **「网站」** 或 **「APP」** 都行（选网站，填 freenav.net）。
4. 保存后会生成一个 **PID**（形如 `mm_12345678_12345_12345`），这就是你的专属 ID。

### 第 3 步：挑商品、拿推广链接
1. 在 App 里搜你想推的商品，比如「机械键盘」「固态硬盘」「极客时间课程」。
2. 点商品 → **「立即推广」** → 选刚才建的推广位 `FreeNav首页`。
3. 复制生成的 **推广链接**（长长一串，里面带你的 PID）。
4. 这个链接就是你要填到站点的东西。

### 第 4 步：填到站点的"插槽"里
打开 `assets/js/config.js`，找到 `FREENAV_AFFILIATE` 这一段，按示例填：

```js
window.FREENAV_AFFILIATE = [
  {
    name: "机械键盘推荐",          // 显示给用户的名字
    note: "刚需外设，成交率高",     // 一句推荐语
    url: "这里粘贴你复制的淘宝联盟推广链接",
    tag: "推广"                    // 合规标识，必须保留
  }
];
```

> 想加几个就加几个（数组里每行一个，逗号隔开）。**留空 `[]` 就不显示。**

### 第 5 步：上线 + 验证
1. 保存文件，推送到 GitHub（和之前一样 `git add . && git commit && git push`）。
2. 等 1 分钟，打开 freenav.net，首页底部会出现「编辑推荐」卡片。
3. 点一下卡片，应该跳到淘宝且地址里带你的 PID——说明成了。

## 四、佣金怎么算 / 怎么提现

- 佣金比例：不同商品不同，一般在 **1%–50%**（数码低、课程/虚拟高）。
- 结算：用户确认收货后，佣金进联盟账户，**每月 20 日左右** 可提现到支付宝。
- 注意：用户**下单后 15 分钟内**没付款，或**退货**，佣金会取消——正常损耗，别慌。

---

## 五、合规提醒（必须看）

- 推广链接**必须标注「推广 / 赞助」**（站点已自动加 `tag: "推广"` 徽章，别删）。
- 别堆太多，破坏"开源 / 免费"的站调。每页 1–3 个足够。
- 优先推和"开源 / 效率工具"调性一致的产品。

## 七、常见问题

**Q：我链接填了但首页没显示？**
A：检查 config.js 里 `FREENAV_AFFILIATE` 数组格式（逗号、引号、方括号）。空 `[]` 不显示。

**Q：用户从阿里云/京东买，淘宝联盟链接能算我佣金吗？**
A：不能。每个平台链接只算该平台的单。想推阿里云就注册阿里云推广，拿它的链接。

**Q：会不会违法？**
A：合法。CPS 是主流变现方式，只要标注「推广」就合规。
